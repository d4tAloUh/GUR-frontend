# GUR (Glovo-Uber-Raketa sample)
####Built with SSR Nuxt
####Ready to deploy to heroku/vercel etc.
## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
